### 线性算法
  - 定义: y= ax + b 
  - 最优解：
    - y: Actural value 真实值
    - Predicted value 预测值，将x带入后得出来表示一般用^y表示
    - Error： 误差，预测值和真实值的差距，一般用e 表示
    - 最优解： 尽可能的找到一个模型使得整体的误差最小，整体的误差通常叫做损失Loss。
    - Loss: 真贵的误差，Loss通过损失函数Loss function计算得到
  - 多元线性回归：
    - 现实生活中往往是多个因素影响y的结果
    - 所以就有了： y = w1X1 + w2X2 + w3X3 + .... + b
    - b是截距，也可以用 wn * 1
    - 使用向量来表示，X表示所有的变量，是一位向量。W表示所有的函数，可以根据向量乘积规律来写：
      - y = W^T X = w1X1 + w2X2 + w3X3 + .... + b
